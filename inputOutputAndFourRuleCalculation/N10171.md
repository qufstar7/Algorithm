# N10171 고양이
## 문제
아래 예제와 같이 고양이를 출력하시오.

## 입력
없음.

## 출력
고양이를 출력한다.

## 예제 입력 1 
## 예제 출력 1 
```
\    /\
 )  ( ')
(  /  )
 \(__)|
 ```

 # 풀이
 ### 답1
 ```java
 public class Main {
    public static void main(String a[]){	
	 
    	System.out.println("\\    /\\"); 
    	System.out.println(" )  ( ')");
    	System.out.println("(  /  )");
    	System.out.println(" \\(__)|");    	   	
     
    }        
}
 ```
### 답2
 ```java
 class Main {
    public static void main(String a[]){	
        System.out.println("\\    /\\\n )  ( ')\n(  /  )\n \\(__)|");   	
     
    }        
}
 ```

 ## 학습)
 * 백슬래쉬 `\`를 문자로 나타내려면 `\\`
